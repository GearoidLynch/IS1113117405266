Product Information

Below there is information on the products we sell


Software as a Service (SaaS) - Salesforce

Software as a services(SaaS) is a software licensing and delivery model in which software is licensed on a subscription basis and is centrally hosted. We provide this service to our customers using Salesforce.


Platform as a Service (PaaS) - Cloud 9

Platform as a Service (PaaS) is a category of cloud computing services that provides a platform allowing customers to develop, run, and manage applications without the complexity of building and maintaining the infrastructure typically associated with developing and launching an app. We provide this service to our customers using Cloud9.


Infrastructure as a Service (IaaS) - Amazon

Infrastructure as a service (IaaS) refers to online services that provide high-level APIs (application programming interface) used to dereference various low-level details of underlying network infrastructure like physical computing resources, location, data partitioning, scaling, security, backup etc. We provide this service to our customers using Amazon.


Communication as a Service (CaaS) - Gmail

>Communications as a Service (CaaS) provides Software as a Service (SaaS) for communications. There is no standard specification as to what is included in CaaS. We provide this service to our customers using Gmail.
